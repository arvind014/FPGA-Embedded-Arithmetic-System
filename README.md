Architecture: Designed using Xilinx Vivado (IP Integrator) with a MicroBlaze soft-core processor.

Firmware: Written in C, featuring real-time arithmetic logic, divide-by-zero error handling, and manual display multiplexing.

Hardware Interfacing: Utilized AXI4-Lite protocols to map physical hardware peripherals to the processor's memory space.

Display Control: Implemented time-division multiplexing to manage 7-segment digits, ensuring high-frequency refresh rates for flicker-free operation.
